# Image Forgery Detection Using ELA & Multi-Scale CNN

## Overview

This project presents a deep learning approach for detecting forged digital images using Error Level Analysis (ELA) and a Multi-Scale Convolutional Neural Network (CNN). The model is designed to capture tampering artifacts at different feature scales, improving image forgery detection performance.

---

## Features

- Error Level Analysis (ELA) preprocessing
- Multi-Scale CNN architecture
- Deep Learning for image classification
- Binary classification (Authentic / Forged)
- Experimental evaluation on the CASIA 2.0 dataset
